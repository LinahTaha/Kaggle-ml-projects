# Kaggle ML Projects (CS375)

Three end-to-end machine learning projects built in Python using scikit-learn, XGBoost, and PyTorch.

## Projects

**House Price Prediction** (`kaggle_improved.ipynb`)  
Regression on the Ames Housing dataset. Engineered 10 features, applied feature importance selection, and compared 3 ML models. Evaluated with RMSE.

**Binary Classification** (`kaggle_bonus.ipynb`)  
1000-feature anonymous dataset with 500 training samples. Used L1 logistic regression for automatic feature selection, reducing to -62 features.

**Credit Default Prediction** (`final.ipynb`)  
American Express dataset (50K customers). Built Logistic Regression, Random Forest, and XGBoost models on imbalanced data. Evaluated with ROC-AUC.

## Tools
Python, scikit-learn, XGBoost, Pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebook
